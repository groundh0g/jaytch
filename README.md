# Jaytch

The Jekyll helper. An easier way to blog.

Jekyll 3.x broke all of my blogs, so I decided to revisit the way I map layouts to markdown. This is the result.

**Twitter:** [@GetJaytch](https://twitter.com/GetJaytch)
